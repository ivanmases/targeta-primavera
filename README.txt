
Targeta virtual — Primavera Arquitectura (Ivan Mases Solé)

Fitxers a pujar a GitHub Pages (arrel del repo):
- index.html
- primavera-ivan-mases.vcf
- og-logo-1200x630.jpg

Passos ràpids:
1) Crea un repo públic (p. ex. targeta-primavera).
2) Afegeix aquests 3 fitxers a l'arrel i fes Commit.
3) Settings → Pages → Deploy from a branch → main / (root) → Save.
4) Envia l'URL: https://<USUARI>.github.io/<REPO>/

(Opcional) Substitueix dins d'index.html les URL relatives per absolutes:
<meta property="og:url" content="https://<USUARI>.github.io/<REPO>/">
<meta property="og:image" content="https://<USUARI>.github.io/<REPO>/og-logo-1200x630.jpg">
